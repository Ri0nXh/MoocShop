# MoocShop
> 慕课视频cap老师的电商项目课程为基础，使用Gin框架构建的项目


# 目录结构说明
```
├─configs           # 配置文件目录
├─internal          # 内部目录
│  ├─api            # api接口
│  ├─dao            # dao层，与数据库交互
│  ├─initialize     # 服务器初始化
│  ├─model          # 模型层
│  └─router         # 路由层
├─logs              # 日志目录
└─wiki              # sql等相关目录
```
